# Groupomania-Projet7
Le projet 7 de Groupomania

#INSTRUCTIONS

- Lancez la console dans le dossier backend, tapez la commande "nodemon server"

- Lancez la console dans le dossier frontend, tapez la commande "npm start"

- Tapez 'Y' pour confirmer le lancement du serveur après que la console constate que le backend tourne deja sur le port 3000

- Vous retrouverez l'application, à l'adresse http://localhost/3001/
